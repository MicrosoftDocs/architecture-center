1. Clone, fork, or download the zip file for the [reference architectures][ref-arch-repo] GitHub repository.

2. Install [Azure CLI 2.0](cli/azure/install-azure-cli?view=azure-cli-latest).

3. Install the [Azure building blocks](https://github.com/mspnp/template-building-blocks/wiki/Install-Azure-Building-Blocks) npm package.

   ```bash
   npm install -g @mspnp/azure-building-blocks
   ```

4. From a command prompt, bash prompt, or PowerShell prompt, login to your Azure account by using the command below.

   ```bash
   az login
   ```
